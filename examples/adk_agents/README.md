# Earth Engine enabled ADK agents - Experimental

The agents in this directory are EXPERIMENTAL and may not have been peer
reviewed or rigorously tested. AI models make mistakes. Use with extreme 
caution!

These examples are based on Google's Agent Development Kit (ADK). It is 
[open source](https://github.com/google/adk-python), has 
[excellent documentation](https://adk.dev/),
and lots of [code samples](https://github.com/google/adk-samples).

## Disclaimer

These agent samples are provided for illustrative purposes only and are not intended
for production use. These basic example agents are intended as foundational
starting points for individuals or teams to develop their own agents.

These samples have not been rigorously tested, may contain bugs or limitations, 
and do not include features or optimizations typically required for a production
environment (e.g., robust error handling, security measures, scalability,
performance considerations, comprehensive logging, or advanced configuration
options).

Users are solely responsible for any further development, testing, security
hardening, and deployment of agents based on these samples. We recommend thorough
review, testing, and the implementation of appropriate safeguards before using
any derived agent in a live or critical system.
